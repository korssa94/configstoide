# 🏭 PLC Code Generator Pro Max

Автоматизированный инструмент для генерации кода ПЛК (программных блоков, GVL, ST) на основе конфигураторов ввода/вывода (ТЭ5), карт адресации и конфигураторов сигнализаций. Проект снабжен удобным веб-интерфейсом на базе Streamlit и полностью изолированным окружением.

---

## 🚀 Быстрый старт (Инструкция по развертыванию)

Инструмент спроектирован так, чтобы его установка и запуск на любом компьютере инженера выполнялись буквально «в два клика» без необходимости ручной настройки системных путей и глобальных библиотек Python.

### Шаг 1. Первая установка (Один раз)
1. Склонируйте репозиторий проекта на свой локальный компьютер.
2. В корневой папке проекта найдите и запустите файл **`install.bat`**.

> **Что произойдет в этот момент:** > * Скрипт проверит наличие Python в Windows. Если Python не установлен, скрипт самостоятельно скачает и установит официальный **Python 3.11** через системный менеджер пакетов `winget`.
> * *Если Python устанавливался с нуля, скрипт попросит вас закрыть консоль и запустить `install.bat` повторно для обновления системных переменных.*
> * В корне проекта автоматически создастся изолированное виртуальное окружение в папке `.venv`.
> * Внутри окружения обновятся все компоненты и установятся строго необходимые библиотеки (Streamlit, openpyxl, pandas и др.).

### Шаг 2. Запуск приложения (Повседневный)
Для повседневного запуска генератора используйте файл **`invisible_start.vbs`** в корне проекта.

> **Как это работает:**
> * Сервер Streamlit запускается в скрытом (фоновом) режиме, не раздражая вас постоянно открытыми черными окнами командной строки на панели задач.
> * Скрипт выждет 5–7 секунд, пока сервер инициализируется, и автоматически откроет вкладку генератора в вашем браузере по умолчанию по адресу: `http://localhost:8501`.

### Шаг 3. Остановка сервера (При необходимости)
Если вам необходимо полностью остановить фоновый сервер и высвободить оперативную память, перейдите в папку `.services/` и запустите файл **`stop_server.bat`**. Он принудительно и безопасно завершит процессы Python, привязанные к генератору.
